#  ⚛️ VueX snippet for Vuex , Vue  & Atom Lovers

Install:
```ssh

apm install atom-vuex
```

Doc : [wiki page](https://github.com/code4mk/atom-vuex/wiki)

Mian Features:

  - VueX Snippet

  Main Feature ❤️


  | Name  |  snippet |
  |---|---|
  |  return/access |  vxr- |
  | vuex| vx- |
  | import| vxi- |
  | functions/element| vxe- |

  - Atom-Vuex = 0.2.0
